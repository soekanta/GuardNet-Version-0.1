‚úÖ Status Model

Random Forest model sudah AKTIF dalam Hybrid Detection Mode! Model ini bekerja bersama Logistic Regression menggunakan weighted voting strategy. Hanya menggunakan 22 URL-based features (tanpa content parsing) untuk prediksi cepat.

‚öñÔ∏è Perbandingan Model

üéØ Akurasi

Logistic Regression: Lebih akurat karena menggunakan 50 features (URL + konten halaman), mampu mendeteksi phishing yang lebih kompleks.

üéØ Akurasi

Random Forest: Kurang akurat (hanya 22 URL features), tetapi lebih cepat karena tidak perlu parsing konten halaman.

‚ö° Kecepatan

Logistic Regression: Lebih lambat karena harus fetch dan parse konten HTML untuk ekstraksi 28 features tambahan.

‚ö° Kecepatan

Random Forest: Sangat cepat karena hanya menganalisis URL tanpa perlu fetch konten (instant analysis).

üîß Preprocessing

Logistic Regression: Memerlukan StandardScaler normalization (mean/std scaling) untuk semua 50 features.

üîß Preprocessing

Random Forest: TIDAK memerlukan normalization, langsung menggunakan raw feature values.

üì¶ Teknologi

Logistic Regression: TensorFlow.js (model.json + weights), trained di Python dengan Keras.

üì¶ Teknologi

Random Forest: Pure JavaScript implementation (rf_model.json), ported dari scikit-learn.

üé≤ Smart Calibration

Logistic Regression: Menggunakan TLD trust bonus, pattern detection (forum/e-commerce), dan URL sanitization untuk mengurangi false positives.

üé≤ Smart Calibration

Random Forest: Tidak ada calibration tambahan, hanya raw prediction dari ensemble trees.

üéØ Use Case

Logistic Regression: Primary model - Digunakan untuk final decision di GuardNet karena lebih reliable dan akurat.

üéØ Use Case

Random Forest: Currently disabled - Memerlukan retraining dengan dataset lebih baik sebelum dapat digunakan.

üîÑ Hybrid Voting Strategy

Ketika Random Forest ENABLED, GuardNet menggunakan weighted voting untuk menggabungkan prediksi kedua model:

‚úÖ Models Agree

Jika kedua model setuju (sama-sama phishing atau sama-sama legitimate):

Final Score = LR √ó 70% + RF √ó 30%

‚ö†Ô∏è Models Disagree

Jika model berbeda pendapat, LR dipercaya lebih karena RF memiliki dataset bias:

Final Score = LR √ó 85% + RF √ó 15%

üö® RF Extreme Confidence

Jika RF sangat yakin (>90% atau <10%) tetapi LR moderat, hampir sepenuhnya percaya LR:

Final Score = LR √ó 95% + RF √ó 5%
